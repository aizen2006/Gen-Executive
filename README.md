## ðŸ“‹ Overview

**GenExecutive** is a modern, high-performance landing page for an "AI + Human Ops Studio". It showcases services that build custom AI agents, chatbots, and landing pages while handling administrative needs, allowing executives to stay in "strategy mode".

The project features a sleek, dark-themed UI with smooth animations, designed to convert visitors into clients for high-ticket operational services.

## âœ¨ Features

- **Modern UI/UX**: Dark mode aesthetic with brand-aligned colors (Cyan, Violet, Dark).
- **Responsive Design**: Fully responsive layout optimized for mobile, tablet, and desktop.
- **Interactive Animations**: Powered by `framer-motion` for scroll reveals, parallax effects, and smooth transitions.
- **Component-Based Architecture**: Modular React components for easy maintenance and scalability.
- **Sections Included**:
  - **Hero**: Impactful value proposition with a 3D-style dashboard visualization.
  - **Social Proof**: Trust indicators and client logos.
  - **Solutions Overview**: Card-based grid detailing services (AI Agents, Chatbots, etc.).
  - **Deep Dive**: Detailed breakdown of specific offerings.
  - **Operations Section**: Visualizing the "AI + Human" hybrid model.
  - **How It Works**: Step-by-step process flow.
  - **Use Cases**: Real-world examples of automation.
  - **Results**: Metric-driven success stories.
  - **Pricing**: Clear tiered pricing structure.
  - **FAQ & CTA**: Common questions and final conversion push.

## ðŸš€ Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites

- **Node.js** (v18 or higher recommended)
- **npm** or **yarn**

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/genexecutive.git
   cd genexecutive
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open in Browser**
   Navigate to `http://localhost:3000` (or the port shown in your terminal).

### Building for Production

To build the app for production:

```bash
npm run build
```

To preview the production build:

```bash
npm run preview
```
